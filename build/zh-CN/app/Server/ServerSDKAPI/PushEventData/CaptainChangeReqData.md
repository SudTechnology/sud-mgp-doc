##### CaptainChangeReqData

| 参数名         | 必选  | 类型     | 说明     |
|:------------|:----|:-------|--------|
| captain_uid | 是   | string | 指定队长ID |

##### 请求示例
```
{
    "event": "captain_change",
    "mg_id": "mg_id_1",
    "timestamp": "1657770493152",
    "data": {
        "captain_uid": "uid_1"
    }
}
```
