断开连接

#

简要描述

  • 断开连接（服务端调用）

请求URL

  • 测试环境：接口地址详见获取服务端API配置中bullet_api对象中字段
  • 生产环境：接口地址详见获取服务端API配置中bullet_api对象中字段

请求方式

  • 传输协议：HTTPS
  • 请求方法：POST
  • 参数以body传递，格式为json（见请求示例）
  • Header中需要添加：Content-Type: application/json;charset=utf-8
  • Header中需要添加：认证请求头 Authorization

请求头 [Authorization]

  • 认证请求头Authorization实现方式

请求参数

参数名 必选 类型 说明
room_id 是 string 房间id

请求示例

{
  "room_id": "xxx"
}

返回参数

  • BaseResp
参数名 必选 类型 说明
ret_code 是 int 响应码
ret_msg 是 string 响应信息
data 否 object 响应数据

返回示例

{
  "ret_code": 0,
  "ret_msg": ""
}
